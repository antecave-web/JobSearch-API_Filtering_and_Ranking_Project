# Job Search API Filtering and Ranking Project

Work-in-progress Python project for automatically collecting, filtering, scoring, and ranking job advertisements based on user-defined criteria.

The goal of the project is to reduce the time required to search large numbers of job postings by automating data collection and relevance assessment.

---

## Project Status

🚧 **Work in Progress**

This project is currently under active development and should be viewed as an exploratory software project rather than a finished application.

Current functionality focuses on:

* Retrieving job advertisements from external sources
* Extracting relevant job information
* Keyword-based scoring
* Ranking opportunities by relevance
* Storing results for later review

Future development may include additional ranking methods, improved matching algorithms, and user-interface improvements.

---

## Project Overview

Searching for employment opportunities often involves reviewing hundreds of postings across multiple platforms.

This project aims to automate that process by:

1. Downloading job advertisements through available APIs
2. Extracting relevant information from each posting
3. Comparing postings against predefined criteria
4. Assigning relevance scores
5. Ranking and storing results for review

The system is designed to function as a personal decision-support tool rather than a replacement for human evaluation.
